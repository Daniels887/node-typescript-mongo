## :wrench: Configuration

#### 1. Run docker-compose

```sh
$ docker-compose up
```

## :gift: Others commands

```sh
# Start existing containers
$ docker-compose start

# Down docker-compose
$ docker-compose down

# Stop docker-compose

$ docker-compose stop

# Exec a command
$ docker-compose exec [CONTAINER_NAME] [COMMAND]
```
